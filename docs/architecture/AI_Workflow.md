# AI Workflow

## 1. Introduction
The AI Workflow describes how the platfrom processes every user interaction from the moment a request is submitted until personalized guidance is generated.
Unlike conventional AI chatbots that directly send user prompts to a Large Language Model (LLM), this platform follows a structured multi-stage pipeline combining Machine Learning, Natural Language Processing, Retrieval-Augmented Generation (RAG), and intelligent reasoning modules.
Each stage contributes to producing responses that are personalized, explainable and actionable.

## 2. Workflow Objectives
The AI Workfloe has been designed to achieve the following objectives:
- Understand the user's situation
- Detect emotional state
- Identify user intent
- Retrieve trustworthy knowledge
- Generate personalized guidance
- Encourage self-reflection
- Recommend one realistic action
- Learn from user feedback
- Improve future recommendations
